To launch the simulation: File viewver/app/Main.py

Left click on a terminal           : [PURPLE HIGHLIGHT] plot the value over the variation of a parameter
Ctr + Left click on a terminal     : [PURPLE HIGHLIGHT] tune a parameter
Left click on a box                : [BLUE   HIGHLIGHT] Move the box

Right click                        : Move
Mouse roll                         : Zoom

All the models are described in the file 'Physics.py'
The connexion between the block are defined in the netlist 'Schematic.py'
